The Code

                    
//get the start and end numbers from the inputs
//Entry point AKA Controller F(x)
function getValues() {
  let startValue = document.getElementById('startValue').value;
  let endValue = document.getElementById('endValue').value;

  let startNumber = parseInt(startValue);
  let endNumber = parseInt(endValue);

  if (Number.isInteger(startNumber) && Number.isInteger(endNumber)) {

    let numberArray = generateNumbers(startNumber, endNumber);
    displayNumbers(numberArray);

  } else {
    Swal.fire({
      icon:'error',
      title: 'Oops!',
      text: 'Please enter a valid number for the start and end values'
    });
  }

  let numberArray = generateNumber(startNumber, endNumber);
  displayNumbers(numberArray);
} 

//Generate the range of numbers to display
//Business/logic F(x)
function generateNumber(start, end) {
  let basketOfNumbers = [];

  for (let number = start; number <= end; number++) {
    basketOfNumbers.push(number); 
  }

  return basketOfNumbers;// => [0, 1, 2... 100]

}

//shows the generated numbers on the page
//and bolds even numbers (aka view function)
function displayNumbers(numbers) {
  
  let results = ''; 

  for (let index = 0; index < numbers.length; index++) {
    let currentNumber = numbers[index];


    if (currentNumber % 2 == 0) {
    results += `<tr><td class="evenNumber">${currentNumber}</td></tr>`;
  } else {
    results += `<tr><td>${currentNumber}</td></tr>`;
  }
}

  let tableBody = document.getElementById('results');
  tableBody.innerHTML = results; 

}
The purpose of this code is to create a functioning web-page that allows for anyone to choose a set of numbers, then to see what numbers are even or odd within it.

How it works...

Our code is divided into 3 seperate sections. The getValues function, the generateNumber function, and the displayNumbers function (You can find the beginning of each section by the word "function"). I have ordered them by what is happening with the code to make it easier to understand.

To begin, we need to know what the function is designed for. The getValues funstion is created to gather data from our app.html page and place the information into our JavaScript. Then it assigns each value from the html into the JavaScript so that we are able to recieve information from the rest of our JavaScript code. Then, we ask the code to go to the next two sections to be understood. First we ask it to be solved by our generateNumber, then we ask it to display on our web-page with displayNumbers.

Once our values are assigned into JavaScript and we have asked it to go to generateNumber, the next function begins. This function asks our code to complete certain properties. It asks for the beginning and end numbers, by default these are assigned to 0 and 100, but can be changed thanks to our HTML. At which point it gathers every number in between and including our numbers by taking the starting number, holding onto that number, then adding 1 to it for the next number. It continues to add these numbers by one, and holding onto each number until it reaches a number that exceeds our maximum value. At which point it stops running, and returns every gathered value that it held onto back to our getValues function. Then the getValues function asks us to take those values and send them to displayNumbers to be put onto the webpage.

Lastly the displayNumbers function recieves every number and places them onto our table in the webpage. But first we created a line of code that told it to take every that is divisble by 2 without leaving a remainder (even numbers) and make them bold on the web-page. Now every number is displayed with even numbers boldened!

Additionally, I have added a notice so that if someone were to use a browser that allowed them to put non-number characters into the number placement locations (where the default 0 and 100 are loacted) then they would get an error asking them to use a valid number in the box.
